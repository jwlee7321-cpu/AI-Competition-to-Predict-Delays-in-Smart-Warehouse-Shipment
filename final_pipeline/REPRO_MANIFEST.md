# Reproduction Manifest

This manifest records the files and weights behind the current final candidates.
All listed training stages use train targets only.  Test rows are used only for
feature construction at inference and for writing submission files.

### `v12_oof_future_blender.py`

Artifact required:

- `v12_full_oof_bundle.npz`, created by the patched `07_v12_full.py`

Purpose:

- rebuild v12 train OOF,
- learn v12/future blend weights on train OOF only,
- test several OOF/test-scale variants, including the successful v28 convention
  where future OOF is raw but future test predictions are mean-scaled,
- learn timeslot, prediction-bin, layout, and layout-cluster weights by
  cross-fitting on train scenarios.

Main outputs:

- `v12_oof_future_blender_report.csv`
- `sub_v38_*`

Run used for the current v38 candidates:

```powershell
python final_pipeline/v12_oof_future_blender.py --prefix sub_v38old --v12-pred-csv sub_v12_main_pre_rerun.csv
```

Key result:

- `sub_v38old_foofraw_predscale_w70.csv` reproduces
  `sub_port_v28_w70.csv` with only numerical roundoff-level differences.
- Best train-OOF dynamic rows are prediction-bin variants around
  `v12raw_ftestscale` and `v12raw_ftrainmean`; mean-calibrated submission probes
  were generated afterward:
  - `sub_v38old_v12raw_ftrainmean_predbin5_mcal.csv`
  - `sub_v38old_v12raw_ftrainmean_predbin10_mcal.csv`

## Late Public Probe: A120

Current late-stage public candidate:

- `sub_v44_anti_v41layout035_a120.csv`: public `9.6773143392`

Formula:

- `A120 = sub_port_v28_w70 - 1.20 * (sub_v41_rank_layout_a035_add_w65 - sub_port_v28_w70)`
- The output is clipped at zero and mean-centered back to the `sub_port_v28_w70`
  prediction mean by `final_pipeline/late_probe_postprocess.py`.

Required scripts after the v28/w70 base exists:

```powershell
python final_pipeline/w70_residual_calibration.py
python final_pipeline/w70_combo_candidates.py
python final_pipeline/late_probe_postprocess.py
```

Roles:

- `w70_residual_calibration.py`
  - uses `v12_full_oof_bundle.npz`, `final_future_stack_oof.npz`, and train
    targets to learn train-OOF-only residual curves around `sub_port_v28_w70.csv`
  - emits `sub_v40_w70cal_rank_layout_a035.csv`
- `w70_combo_candidates.py`
  - combines the v40 residual shape with the v28-line `w65` direction
  - emits `sub_v39_v28line_w065.csv` if needed
  - emits `sub_v41_rank_layout_a035_add_w65.csv`
- `late_probe_postprocess.py`
  - reverses the public-failed v41 direction
  - emits `sub_v44_anti_v41layout035_a120.csv`

Exact reproduction note:

- `sub_port_v28_w70.csv` was created from the pre-rerun v12 prediction.  Preserve
  `sub_v12_main_pre_rerun.csv` for exact reproduction of the current public
  chain.  A fresh `07_v12_full.py` rerun can create a slightly different
  `sub_v12_main.csv`, so raw retraining without this preserved prediction may
  not be bit-identical.
